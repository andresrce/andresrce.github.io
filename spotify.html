Top 200 2017-2023 Spotify

Este trabajo implica la realización de un Análisis Exploratorio de Datos (EDA) en un conjunto de datos de Spotify utilizando Python. El enfoque se centra en comprender las características de las canciones, las nacionalidades de los artistas y las tendencias temporales en las preferencias musicales. El estudio se desarrolla principalmente en torno al análisis exploratorio, seguido de la aplicación de modelos de regresión lineal que incorporan variables categóricas y características de las canciones. Además, se lleva a cabo un análisis conciso de datos de series temporales.

El proceso detallado del proyecto consistió en el uso de Python y diversas bibliotecas, las cuales facilitaron la limpieza de datos, la visualización y el trabajo con modelos de regresión y análisis de series de tiempo.
